# Python旅游分析推荐系统 python776

本项目是一款集旅游数据分析与景点推荐于一体的系统，旨在为旅游数据研究者和爱好者提供全面的功能支持。

## 技术栈

- 后端：Python + Django
- 数据库：MySQL
- 前端可视化：ECharts

## 功能模块

### 用户系统

- 支持登录/注册功能
- 支持个人信息和密码的修改

### 数据展示

- 首页展示景点标签数量、评分TOP10、销量TOP10、地理分布等信息

### 评论管理

- 支持景点评论的添加和评分

### 智能推荐

- 基于用户协同过滤的景点推荐

### 数据分析

- 多维度旅游信息分析
- 可视化图表展示

## 系统角色

- 旅游数据研究者
- 旅游数据爱好者

## 运行环境

- 推荐使用Python 3.x版本
- Django 2.x版本
- MySQL数据库

## 部署步骤

1. 环境准备：安装Python、MySQL和Django等依赖库
2. 数据库配置：创建MySQL数据库和表结构
3. 代码部署：部署后端Python代码和前端页面
4. 系统启动：运行Django项目，使系统处于可用状态
